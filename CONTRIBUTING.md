# Contributing to TermUI

Thank you for your interest in contributing to TermUI! This document will guide you through the process.

## Prerequisites

Before you start, make sure you have:

| Requirement | Version | Install/Check |
|-------------|---------|---------------|
| **Bun** | >= 1.3.0 | [Install Bun](https://bun.sh/docs/installation) → `bun --version` |
| **Node.js** | >= 18.0.0 | [Install Node.js](https://nodejs.org/) → `node --version` |
| **Git** | Latest | `git --version` |

## Getting Started

### 1. Fork the Repository

Click the **Fork** button on the top right of the [TermUI repo](https://github.com/Karanjot786/TermUI).

git clone https://github.com/YOUR_USERNAME/TermUI.git
